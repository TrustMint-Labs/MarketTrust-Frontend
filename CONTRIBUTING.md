# Contributing to MarketTrust

Thanks for taking the time to contribute! 🎉

Please read the full **[Contributing Guide](./docs/CONTRIBUTING.md)** before opening a pull request or issue. It covers:

- [Development setup](./docs/CONTRIBUTING.md#development-setup)
- [Project conventions](./docs/CONTRIBUTING.md#project-conventions) (TypeScript, naming, styling, commits)
- [Pull request process](./docs/CONTRIBUTING.md#pull-request-process)
- [Adding UI components](./docs/CONTRIBUTING.md#adding-a-new-ui-component)
- [Adding escrow actions](./docs/CONTRIBUTING.md#adding-a-new-escrow-action)

For security vulnerabilities, see [SECURITY.md](./docs/SECURITY.md) — **do not open a public issue**.
